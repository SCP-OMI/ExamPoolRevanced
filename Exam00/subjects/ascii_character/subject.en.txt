Assignment name  : ascii_character
Expected files   : ascii_character.c
Allowed functions:
Version          : 1
--------------------------------------------------------------------------------

Write a function that returns ASCII character of the character passed to it.

Your function must be declared as follows:
    int ascii_character(char c):

Example:
    $> ascii_character('A') = 65
    $> ascii_character('z') = 122
    $> ascii_character('B') = 66
