Assignment name  : prev_character
Expected files   : prev_character.c
Allowed functions: 
Version          : 1
--------------------------------------------------------------------------------

Write a function that  returns the previous character followed by a newline ("\n").

Your function must be declared as follows:
    char prev_character(char c);

Example:
    $>  prev_character ('B') = A
    $>  prev_character('a') = z
