Assignment name  : ft_strjoin
Expected files   : ft_strjoin.c
Allowed functions: malloc, free
Version          : 1
--------------------------------------------------------------------------------

Implement the ft_strjoin function, which concatenates two strings into a new one.

The function must return a pointer to the new string, which is the concatenation of s1 and s2.
    - If memory allocation fails, the function must return NULL.

Your function must be declared as follows:
    char *ft_strjoin(char const *s1, char const *s2);
