Assignment name  : ascii_character
Expected files   : ascii_character.c
Allowed functions: write
Version          : 2
--------------------------------------------------------------------------------

Write a program that displays an ASCII character followed by a newline ("\n").

If the number of arguments is not 1, display only a newline.

Example:

$> ./ascii_character 'A' | cat -e
65$
$> ./ascii_character 'z' | cat -e
122$
$>./ascii_character | cat -e
$
$>./ascii_character "" | cat -e
